# 故障排查

## Q1：执行 `npm run dev` 提示 `'vite' 不是内部或外部命令`

```
'vite' 不是内部或外部命令，也不是可运行的程序或批处理文件。
```

**原因**：系统环境变量 `NODE_ENV=production`，npm 在 production 模式下跳过 `devDependencies` 安装，导致 vite 等开发依赖未安装。

**解决方案**：

```bash
npm install --include=dev
```

项目根目录已添加 `.npmrc` 文件，内容为 `include=dev`，后续 `npm install` 会自动包含 devDependencies。
